# 다풀자 단어장
`인사이터 / 다풀자 팀의 프론트엔드 과제 B`에 해당합니다. React를 활용해 간단한 단어장 어플리케이션을 구현합니다. 소스코드 내에 주석처리된 TODO를 완료해 어플리케이션을 완성합니다.

# 주요기능

- 단어 목록을 볼 수 있습니다.
- 목록에 있는 단어를 활용해 3지 선다형 뜻 찾기 문제를 풀 수 있습니다.
